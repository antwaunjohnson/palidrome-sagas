                              function palindromeCheck(userString) {
                                userString = userString.toLowerCase();
                              
                                let regex = /[^a-z0-9]/gi;
                                userString = userString.replace(regex, "");
                              
                                let revString = [];
                                let returnObj = {};
                              
                                for (let index = userString.length - 1; index >= 0; index--) {
                                  revString += userString[index];
                                }
                              
                                if (revString === userString) {
                                  returnObj.msg =
                                    "You have completed your journey through the Palindrome Sagas!";
                                } else {
                                  returnObj.msg = " Your journey through the Palindrome Sagas must continue!";
                                }
                              
                                returnObj.reversed = revString;
                              
                                return returnObj;
                              }

                                document.getElementById("alert").classList.remove("invisible");

The code is structured in three functions

getValue

The getValue function begins with using the document method getElementById to find the element with the "alert" id and add the class of "invisible" to it. The variable userString calls on the document method getElementById to find the element with the id of "userString" from the input element. The value property is appended to document method getElementById to get the value contained within the input element.

The variable returnObj is set to the value of the palindromeCheck function which takes the userString as an argument. Finally the displayMessage is called which takes in the returnObjvariable as an argument

palindromeCheck

The palindromeCheck function begins by using the toLowerCase method ensure that the string entered by the user is converted to lowercase characters. Next the regex variable is set to a regular expression that removes all spaces and special characters from the string. The replace method is called on the userString. You can learn more about the replace method here

The next part of the of the function has two variables revString, which is set to an empty array and returnObj, which is set to an empty object. A for loop is used to loop through the revString array. Within the for loop index is first set to the length of the userString - 1 to begin at the last element. As long as the index is greater than or equal to 0, the count is decremented. We use revString += userString[index] to return our string reversed one character at a time.

Once we have the reversed string we use the if statement to determine if the revString is equal to the userString. If the check is successful, the user will receive a success message. If the check is unsuccessful, the user is encouraged to try again.

To conclude, the reversed is called on returnObj and set equal to revString. Finally, we return returnObj.

displayMessage

The displayMessage function begins by using the getElementById document method to find the element with the id of alertHeader and set it equal to returnObj.msg. We use the same document again to find the element with the id of message. The element property innerHTML is chained to the getElementById method to provide access to change the HTML within the element. Template literals (Template strings) are used to ensure that the message the user receives is dynamic. Finally, we find the id of alert using the getElementById method to remove the class of invisible from the element.
